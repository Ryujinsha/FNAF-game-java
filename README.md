Java OOP Game Learning Project

About this project
This repository contains my learning project where I study Java Object Oriented Programming by building a simple game. The project is part of my practice to understand how game logic, class structure, and basic systems can be built using Java.

The game concept is inspired by the mechanics of Five Nights at Freddy's, especially the survival gameplay and random events that create tension for the player. The goal is not to recreate the original game, but to learn how similar systems can be implemented using Java.

This project is also part of my learning process using AI as a programming assistant to help explore ideas, debug code, and understand programming concepts better.

Learning goals
Learn Object Oriented Programming in Java
Practice designing game logic
Understand random event systems using RNG
Improve problem solving skills in programming
Experiment with AI assisted development

Technologies used
Java
Object Oriented Programming concepts
VS Code
Git and GitHub

Game concept
The game focuses on survival mechanics where the player must monitor situations and respond to random events generated by the system. Enemy behavior and events may use random logic to simulate unpredictable gameplay.
